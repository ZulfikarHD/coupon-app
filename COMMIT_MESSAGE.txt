## Technical Details

### Phone Normalization
- Automatically converts various formats to 628xx
- Supports: 08xx, +628xx, 628xx, 8xx formats
- Display format: 0812-3456-7890

### Coupon Code Format
- Format: ABC-1234-XYZ
- Guaranteed uniqueness
- Generated via Job or helper function

### Database Schema
- Proper indexes on code, status, phone, dates
- Foreign key relationships
- Enum types for status and action
